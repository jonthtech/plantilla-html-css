# Proyecto Base: Capacitación HTML5 y CSS3 (Trayecto 2)

¡Bienvenido a tu primer proyecto de maquetación web! 🚀

## Objetivo del Ejercicio
En este repositorio encontrarás la estructura básica de un portafolio profesional. Tu tarea durante las próximas 6 semanas será:
1. Personalizar el contenido del archivo `index.html`.
2. Aplicar estilos avanzados en `style.css` usando Flexbox y Grid.
3. Asegurar que la página sea responsiva (adaptable a móviles).
4. Realizar commits constantes para documentar tu progreso en GitHub.

## Instrucciones Rápidas
- Abre este proyecto en **Visual Studio Code**.
- Usa la extensión **Live Server** para ver los cambios en tiempo real.
- No olvides sincronizar tus cambios con el comando `git push`.

¡Mucho éxito, futuro desarrollador!
